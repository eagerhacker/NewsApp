# NewsApp
_Search news and blog articles from NewsAPI_

# Features:
+ MVVM architecture.
+ Retrofit network library.
+ Hilt for dependency injection.
+ Navigation Components.
+ Offline database cache using RemoteMediator.
+ Kotlin Coroutines.
+ LiveData, ViewModel, ViewBinding.
